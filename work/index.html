Memento / 2000

時系列が逆転して進む、クリストファー・ノーラン監督にとっての出世作。

10分間しか記憶が残らない障害を負った主人公が、妻を殺した犯人を探し出す。
主人公はメモやポラロイドで記録した重要な情報と、障害の持つ前の少ない記憶を頼りに謎を解き明かしていく。

ダークナイト / 2008

アメコミ映画の歴史を変えたとも称される伝説的映画。

ダークナイト三部作の2作目にあたる。バットマンは市警や地方検事のハーヴィーとともに
悪のカリスマ、ジョーカーと対峙する。 ジョーカーの度重なる攻撃にいくつもの苦渋の決断を迫られる。アクションの枠を超えた猟奇的クライムサーガ。
ラストシーンのカッコ良さは異常。

インセプション / 2010

夢と現実が交差した新感覚映画

人の夢に入りこみ記憶を抜き取る企業スパイのコブは、競争相手の企業を解体させるべくインセプション(意識の改算)を行う。 コブは優秀なメンバーを集め、この不可能なミッションに挑戦する。しかし夢の中では思いがけない相手が待ち受けていいた。 ラストシーンの謎に注目。 今敏監督の映画「パプリカ」からインスピレーションを受けたと考えられ、共通点などを考察するのも面白い。

インターステラー / 2014

人類の存亡と親子の愛を描いたSF超大作。

近未来の地球、環境汚染や硫黄気象により人類の生存が不可能になりつつあった。元宇宙飛行士のマーフは子供たちを残し、人類のため遥かな宇宙へ旅立った。 ブラックホールや重力、高次元世界など科学的考証に基づいた臨場感のある映像となっている。 アメリア(アン・ハサウェイ)の「愛には特別な意味がある。私たちはまだ理解していないだけ。」という言葉に注目。

TENET / 2020

ノーラン監督によるまったく新しい映像体験。

「インセプション」、「インターステラー」を超える超難解映画。逆行する時間のなか第三次世界大戦を防ぐため、未来に関する重要な鍵を持つ武器商人、セイターの陰謀に立ち向かう。 映像、ストーリーなど全てが観客の想像を超える超大作アクション。1度の鑑賞では理解することができないので2度鑑賞することを強くおすすめします。 冒頭シーンで主人公を助けた謎の兵士のリュックに注目。
